# Pr 9 Data Visualization

# **Overview**

This project is a **Sales Data Analysis and Visualization program** developed using Python.
The program allows users to load a sales dataset, explore the data, perform different DataFrame operations, handle missing values, generate descriptive statistics, and create various visualizations.

It is a **menu-driven application** that demonstrates the use of Pandas, NumPy, Matplotlib, and Seaborn for data analysis and graphical representation.

---

# **Objectives**

* To understand **data analysis using Python**
* To perform **data manipulation using Pandas**
* To use **NumPy for numerical operations**
* To generate **data visualizations**
* To build a **menu-driven analysis program**

---

# **Feature**

* Load dataset from CSV file
* Explore dataset information
* Search, filter, and sort data
* Perform DataFrame operations
* Handle missing values
* Generate descriptive statistics
* Create multiple data visualizations
* Save visualization as image file

---

# **Functions**

The main functions used in the program:

* `load_data()` → Load dataset from CSV file
* `explore_data()` → Display dataset information
* `dataframe_operations()` → Perform Pandas and NumPy operations
* `handle_missing()` → Handle missing values in dataset
* `descriptive_stats()` → Generate statistical summary
* `visualize()` → Create different charts
* `save_visualization()` → Save generated chart
* `menu()` → Run the menu-driven program
